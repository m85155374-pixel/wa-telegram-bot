# 📱 بوت الرسايل المحذوفة من واتساب ← تيليجرام

## ⚙️ خطوات التشغيل

### 1. افتح الملف `index.js` وحط التوكن بتاعك:
```js
const TELEGRAM_TOKEN = 'حط_التوكن_هنا';
```

### 2. ثبّت المتطلبات:
```bash
npm install
```

### 3. شغّل البوت:
```bash
node index.js
```

### 4. هيظهرلك QR code في التيرمنال:
- افتح واتساب على موبايلك
- اضغط **النقاط الثلاث** ← **الأجهزة المرتبطة**
- امسح الـ QR code

### 5. ✅ خلاص! أي رسالة تتمسح هتوصلك على تيليجرام

---

## 📋 متطلبات التشغيل
- Node.js v18+
- Chrome أو Chromium مثبت على السيرفر
- اتصال إنترنت مستمر

## ☁️ للتشغيل على Railway:
1. ارفع الملفات على GitHub
2. افتح railway.app وعمل New Project من GitHub
3. Railway هيشيل كل حاجة أوتوماتيك

## ⚠️ ملاحظات
- لازم البوت يشتغل 24/7 على سيرفر
- واتساب لازم يكون متوصل دايماً
- الصور بتتبعت لو اتعملت cache قبل المسح
